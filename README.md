# About Our Program:
### Features;
- Text
- Accent Detection
    - Audio File -> Convert to text 
    - Microphone -> Convert to text
<hr>

# Dependencies:
### Set up;
- we use OpenAI to generate an Answer to the text
    - `conda install conda-forge::openai`
- Text to Speech
    - `pip install SpeechRecognition`
